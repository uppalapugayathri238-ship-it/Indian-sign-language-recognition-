# Indian Sign Language Recognition using Quantum Machine Learning

## Overview
This project recognizes Indian Sign Language (ISL) hand gestures using Quantum Machine Learning (QCNN). It uses MediaPipe for hand landmark detection and TensorFlow for preprocessing.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- TensorFlow
- PennyLane
- NumPy

## Features
- Real-time hand gesture recognition
- Quantum Machine Learning (QCNN)
- Live camera detection
- Accuracy evaluation
